D&D Character Creation Web App

Here a webapp for a Dungeons and Dragons chacacter maker. He you can see you can see the tech stack of the samples tailored in the webapp.

Tech Stack:
  • 1. Frontend
    • React.js: For building a dynamic, interactive user interface with reusable components.
    • Chakra UI: Provides accessible, customizable UI components perfect for forms and layout.
  • 2. Backend
    • Node.js with Express.js: Handles API routes and business logic for character creation and data management.
  • 3. Database
    • A relational database to store user data, character details, and reference tables like spells, items, and races.
  • 4. Authentication & Authorization:
    • Auth0: Offers a secure and scalable solution for user login, including social media authentication.
  • 5. File Storage
    • AWS S3: For storing user-uploaded avatars and generated PDFs of character sheets.
  • 6. Deployment & Hosting
    • Vercel: Simplifies frontend deployment with continuous integration and easy previews.
    • Backend hosting and resource scaling management.
