D&D Dungeon Map Creator

Here is another webapp for a Dungeons and Dragons fans to create thier very own dungeon and share. He you can see you can see the tech stack of the samples tailored in the webapp.

Tech Stack:
  • 1. Frontend & Visualization
    • Vue.js: A lightweight framework for creating an interactive interface where users can build, view, and customize dungeon maps.
  • 2. Backend & Map Generation
    • Python (Flask): Handles procedural map generation using libraries like networkx or Dungeon Crawl and serves map data through APIs.
  • 3. Storage & Hosting
    • Firebase: Provides real-time database storage for map data, along with hosting for frontend assets and user-generated content.
